# MonProjet

programme C pour le CNN

## Structure du Projet
```
cnn_riscv_mancini/
├── CMakeLists.txt
├── src/
├── include/
├── build/
└── bin/
```
## Compilation : 

1. **Créer un répertoire de build :**
    ```
    mkdir build
    cd build
    ```

2. **Générer les fichiers de construction avec CMake :**
    ```
    cmake ..
    ```

3. **Compiler et executer :**
    ```
    make
    ../bin/CNN
    ```
